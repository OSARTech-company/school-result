CSV Format (Scores Upload)

Use this format to upload scores for existing students in your assigned class:

  • Student ID - Existing student ID
  • Subject - One subject per row
  • Test 1 ... Test {{ max_tests }} (combined test total must stay within school test maximum)
    • {% if exam_mode_view == 'combined' %}
  • Exam Score (your class/classes use combined exam mode)
    • {% elif exam_mode_view == 'separate' %}
  • Objective and Theory (your class/classes use separate exam mode)
    • {% elif exam_mode_view == 'mixed' %}
  • Objective and Theory for separate-mode classes
  • Exam Score for combined-mode classes
    • {% endif %}
  • Teacher Comment (optional)
